# Hidden_chat

This is a simple web app - chat rooms.
You can join the chat and communicate with other participants online.
Nobody knows who you are:)

Python web application implemented using WebSockets technology.
# Quick start

It is necessary to fill in .env<br>
then:<br>

pip install -r requirements.txt<br>
docker compose down<br>
docker compose up -d --build<br>
